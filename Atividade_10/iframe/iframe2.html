JavaServer Faces Stands for "JavaServer Faces." JSF is a framework that allows Web developers to build user interfaces for JavaServer applications. ... JSF simplifies the creation of Web applications by providing a standard set of tools (or an API) for building user interfaces
JDO Java Data Objects (JDO) is an application program interface (API) that enables a Java programmer to access a database implicitly - that is, without having to make explicit Structured Query Language (SQL) statements. ... An alternative to JDO and JDBC is Enterprise JavaBeans (EJB).
Hibernate Hibernate is an Object-Relational Mapping (ORM) solution for JAVA. It is an open source persistent framework created by Gavin King in 2001. It is a powerful, high performance Object-Relational Persistence and Query service for any Java Application.
